HTML5 웹 프로그래밍 실습

2022년 07월 12일

<실습현장>



서울IT아카데미에서 정보처리산업기사(과정평가형)수업을 통해 HTML5 웹 프로그래밍 실습을 하고있다.      수업을 통해 보고 배운것을 이용해 나만의 최종 포트폴리오를 만드는것이 목표이다.


웹 페이지 기본 구조와 작성 방법

2022년 07월 05일


1. 태그와 요소


HTML페이지를 구성하는 각 부품을 '요소'라고 한다. 태그는 이러한 요소를 만들때 사용하는 작성 방법이다.


2. 속성


속성은 태그에 추가 정보를 부여할 때 사용하는 것이다.

<h1 title="header">Hello HTML5</h1>
<img src="image.png">


3. 주석


프로그래밍 언어에서 프로그램 실행에 영향을 미치지 않고 설명용으로 사용하는 코드를 '주석'이라고 한다.

<!-- 주석 -->


4. 내부 스타일시트 작성과 실행


<style>
h1{
color: white;
background: black;
}
</style>


5. 외부 스타일시트 작성과 실행


스타일시트에 다음과 같이 코드를 작성하고 Style.css로 저장한다.
h1{
color: white;
background: black;
}

그런 다음 link 태그를 사용해 외부 스타일시트를 불러온다.
<head>
<link rel="stylesheet" href="Style.css">
</head>


6. 내부 자바스크립트 작성과 실행


head 태그나 body 태그 내부에 다음과 같이 script 태그를 생성한다.

<script>
alert('Hello JavaScript');
</script>


7. 외부 자바스크립트 작성과 실행


다음과 같이 코드를 작성하고 파일명.js로 저장한다.
alert('OuterScript');

그리고 script 태그를 사용해 외부 자바스크립트를 불러온다.
<script src="파일명.js"></script>

HTML5 기본 태그

2022년 07월 06일


1. 제목과 본문 글자 태그


태그 설명
제목 글자 h1 첫 번째로 큰 제목 글자 생성
h2 두 번째로 큰 제목 글자 생성
h3 세 번째로 큰 제목 글자 생성
h4 네 번째로 큰 제목 글자 생성
h5 다섯 번째로 큰 제목 글자 생성
h6 여섯 번째로 큰 제목 글자 생성
본문 글자 p 본문 문단 생성
br 줄 바꿈
hr 수평 줄 사입

공백, 괄호 등은 다음 특수 문자를 사용해 화면에 표시한다.
특수 문자 출력 문자
& nbsp; 공백
lt; <
gt; >
amp; &

2. 앵커 태그


a 태그는 다른 웹 페이지나 웹 페이지 내부의 특정 위치로 이동할 때 사용한다.
<a href="http://이동할 웹 페이지">출력글자</a>

웹 페이지 내부의 특정 태그로 이동하려면 id 속성을 추가로 사용한다.
<a href="alpha">Alpha 부분</a>
<h1 id="alpha">Alpha</h1>


3. 글자 모양 태그


태그 설명
b 굵은 글자
i 기울어진 글자
small 작은 글자
sub 아래 첨자
sup 위 첨자
ins 밑줄 글자
del 취소선이 그어진 글자

4. 목록 태그


태그 설명
ul 순서가 없는 목록
ol 순서가 있는 목록
li 목록 요소 생성

5. 테이블 태그


태그 설명
table 표 삽입
tr 행 삽입
th 제목 셀 생성
td 일반 셀 생성
>
태그 속성 설명
table border 표의 테두리 두께 지정
th,td colspan 셀의 너비 지정
rowspan 셀의 높이 지정

6. 미디어 태그


태그 속성 설명
img 태그 src 이미지의 경로 지정
alt 이미지가 없을 때 나오는 글자 지정
width 이미지의 너비 지정
height 이미지의 높이 지정
audio, video 태그 src 음악, 비디오 파일의 경로 지정
preload 음악, 비디오를 준비 중일 때 데이터를 모두 불러올지 여부 지정
autoplay 음악, 비디오의 자동 재생 여부 지정
loop 음악, 비디오의 반복 여부 지정
controls 음악, 비디오 재생 도구 출력 여부 지정
video 태그 width 비디오의 너비 지정
height 비디오의 높이 지정


반응형 웹

2022년 07월 12일


반응형 웹은 웹 페이지 하나로 데스크톱, 태블릿PC, 스마트폰에 맞게 디자인이 자동으로 반응해서 변경되는 웹페이지이다.


1. viewport meta 태그 사용하기

viewport meta 태그를 추가함으로써 스마트폰에서도 정상적인 크기로 볼 수 있다.

<head>
<meta name="viewport" content="user-scalable=no, initial-scale=1">
</head>

user-scalable: 화면 및 축소 가능 여부
initial-scale: 초기 확대 비율
